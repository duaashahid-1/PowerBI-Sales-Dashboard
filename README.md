# PowerBI-Sales-Dashboard


## 📊 Project Overview
This project is an interactive **Sales Performance Dashboard** created using **Power BI**.  
It provides insights into sales, profit, and customer behavior across regions and product categories.  

The dataset simulates realistic sales data, including orders, products, customers, and profits. This is ideal for demonstrating Power BI skills in data visualization and analysis.

---

## 🛠️ Tools Used
- **Power BI Desktop** – for creating the interactive dashboard  
- **Excel** – for dataset preparation and storage  

---

## 📁 Dataset
The dataset includes 100 records with the following columns:  
- `Order_ID` – Unique identifier for each order  
- `Order_Date` – Date of the order  
- `Region` – Sales region (North, South, East, West)  
- `Category` – Product category (Electronics, Furniture, Office Supplies)  
- `Product` – Product name  
- `Customer` – Customer name  
- `Sales` – Sales amount  
- `Quantity` – Quantity sold  
- `Profit` – Profit earned  

---

## 📊 Dashboard Features
1. **KPI Cards** – Total Sales, Total Profit, Total Orders, Profit Margin  
2. **Sales by Category** – Bar chart showing category-wise sales  
3. **Sales by Region** – Pie chart showing sales distribution across regions  
4. **Sales Trend Over Time** – Line chart displaying monthly sales  
5. **Top Customers** – Table of highest contributing customers  
6. **Slicers** – Filter data by Region, Category, and Date  

---

## 🔥 Advanced Features
- **Top 5 Products Visual** – Highlights best-selling products  
- **Dynamic Titles** – Dashboard updates title based on selected slicers  
- **Profit Margin Calculation** – DAX measure to calculate profitability  

---

## 🚀 How to Use
1. Open `Sales_Dashboard.pbix` in Power BI Desktop  
2. Explore the interactive visuals  
3. Use slicers to filter data and gain insights  


---

## 📝 Learning Outcomes
- Hands-on experience with Power BI visuals  
- Creating DAX measures for KPIs  
- Building interactive dashboards with slicers and filters  
- Organizing and presenting data for storytelling

---

## 🛠 Tools Used
- SQL (MySQL / SQL Server)
- Excel (for dataset creation)

## 📂 Dataset

  - `order_date` → Date of order

  - `region` → Sales region
