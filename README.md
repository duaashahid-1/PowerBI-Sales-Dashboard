# PowerBI-Sales-Dashboard


## ЁЯУК Project Overview
This project is an interactive **Sales Performance Dashboard** created using **Power BI**.  
It provides insights into sales, profit, and customer behavior across regions and product categories.  

The dataset simulates realistic sales data, including orders, products, customers, and profits. This is ideal for demonstrating Power BI skills in data visualization and analysis.

---

## ЁЯЫая╕П Tools Used
- **Power BI Desktop** тАУ for creating the interactive dashboard  
- **Excel** тАУ for dataset preparation and storage  

---

## ЁЯУБ Dataset
The dataset includes 100 records with the following columns:  
- `Order_ID` тАУ Unique identifier for each order  
- `Order_Date` тАУ Date of the order  
- `Region` тАУ Sales region (North, South, East, West)  
- `Category` тАУ Product category (Electronics, Furniture, Office Supplies)  
- `Product` тАУ Product name  
- `Customer` тАУ Customer name  
- `Sales` тАУ Sales amount  
- `Quantity` тАУ Quantity sold  
- `Profit` тАУ Profit earned  

---

## ЁЯУК Dashboard Features
1. **KPI Cards** тАУ Total Sales, Total Profit, Total Orders, Profit Margin  
2. **Sales by Category** тАУ Bar chart showing category-wise sales  
3. **Sales by Region** тАУ Pie chart showing sales distribution across regions  
4. **Sales Trend Over Time** тАУ Line chart displaying monthly sales  
5. **Top Customers** тАУ Table of highest contributing customers  
6. **Slicers** тАУ Filter data by Region, Category, and Date  

---

## ЁЯФе Advanced Features
- **Top 5 Products Visual** тАУ Highlights best-selling products  
- **Dynamic Titles** тАУ Dashboard updates title based on selected slicers  
- **Profit Margin Calculation** тАУ DAX measure to calculate profitability
